# Expected Policy Questions

A strong PDA trace should make some version of these decision points explicit:

1. Is PiP a free panel or a video-shaped object?
2. What persists across resize/remount?
3. What hit-zone semantics define move vs edge resize vs corner resize?
4. How is click suppressed after drag/resize?
5. Who owns the gesture during active interaction?
6. What counts as sufficient closure before implementation proceeds?
